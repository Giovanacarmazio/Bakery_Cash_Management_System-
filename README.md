# Bakery Cash Management System

Este repositório contém o projeto **Bakery Cash Management System**, que implementa um sistema de gestão de caixa para uma padaria. O sistema permite o registro de vendas, controle de fluxo de caixa e geração de relatórios financeiros, facilitando a gestão financeira de pequenos negócios.

## Descrição do Projeto

O objetivo do **Bakery Cash Management System** é criar uma solução prática para controlar as entradas e saídas de caixa de uma padaria, gerando relatórios de vendas e ajudando na análise de desempenho financeiro.

## Funcionalidades

- Registro de vendas diárias.
- Controle de entradas e saídas financeiras.
- Geração de relatórios de fluxo de caixa.

## Tecnologias Utilizadas

- **Python**
- **SQLite**: Para armazenar os dados das transações.

## Estrutura do Repositório

- `cash_management.py`: Código principal do sistema de gestão de caixa.
- `database.db`: Banco de dados SQLite que armazena as transações financeiras.
- `reports/`: Relatórios financeiros gerados pelo sistema.

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Giovanacarmazio/Gestao-de-Caixa-Padaria.git
   ```

2. Execute o script:
   ```bash
   python cash_management.py
   ```

3. Insira os dados das transações e acompanhe os relatórios no diretório `reports/`.
